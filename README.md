# clj-pub

Desktop EPUB reader written in Clojure.

![Alt text](./screenshot_1.png?raw=true "Title")

# Usage


    $ lein run

# Running with clj

```bash
clj -Sdeps '{:deps {epub {:git/url "https://github.com/netb258/clj-pub.git" :sha "ddb25952afbb4e7e2a58647d72e99adf95e3b65e"}}}' -m epub.core
```
